Guided Tutorials

Step 1: Deploy the Contract

Click the "Deploy Contract Locally" button to deploy the TeachLink contract to a local Stellar network.

Step 2: Initialize the Bridge

Use the API Explorer to call the 'initialize' function with appropriate parameters.

Step 3: Bridge Tokens

Call 'bridge_out' to lock tokens for cross-chain transfer.

Step 4: Complete Bridge

Simulate validator signatures and call 'complete_bridge' to release tokens.

Step 5: Explore Escrow

Create a multi-signature escrow using 'create_escrow'.
