## Interactive Tutorials

For hands-on learning, try our [Interactive Documentation](../interactive/README.md):

```bash
cd docs/interactive
cargo run
```

Then open http://localhost:3000 in your browser.

## Prerequisites

Before starting tutorials, ensure you have:
- Rust toolchain installed
- Stellar CLI (stellar) installed
- A Stellar wallet (Freighter or Albedo)
- Testnet XLM tokens for testing
