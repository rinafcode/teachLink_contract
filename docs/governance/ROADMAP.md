## How to Influence the Roadmap

### 1. Feature Requests
- Open a [feature request](https://github.com/rinafcode/teachLink_contract/issues/new?template=feature_request.yml)
- Include clear use cases and potential impact

### 2. Community Discussions
- Join [GitHub Discussions](https://github.com/rinafcode/teachLink_contract/discussions)
- Participate in Discord conversations

### 3. Proposal Process

For significant changes, submit a TeachLink Improvement Proposal (TIP):

1. **Draft**: Create a draft in Discussions
2. **Feedback**: Gather community input (2 weeks)
3. **Refinement**: Incorporate feedback
4. **Review**: Maintainer review period (1 week)
5. **Decision**: Accept/Reject/Defer

### 4. Voting (Future)
Once governance is established, token holders will vote on major roadmap decisions.
